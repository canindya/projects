02

StockPicker

1,800 stocks in. 50 out.

A six-stage analysis pipeline that scans every stock on NSE — combining fundamental screening, technical signals, quant momentum scoring, and a three-model ML ensemble — to surface the 50 stocks most likely to grow. Includes a full web UI with deep-dive profiles, portfolio builder, swing trade setups, paper trading, and exportable reports.

Python Machine Learning NSE / Indian Markets Quant Full-Stack
Status
Active Development
The Insight
Retail investors drown in stock tips but lack systematic frameworks. StockPicker replaces gut feeling with a transparent, multi-signal scoring engine — every pick comes with a full breakdown of why it scored the way it did.
How It Works
5-layer data pipeline with near-zero gaps → macro regime detection → hard fundamental filters → 9 technical + 7 momentum signals → RF/XGBoost/LR ensemble → risk-adjusted final ranking. All thresholds configurable via YAML.
03

Cost of Living Tracker

193 countries. One dashboard.

An interactive global cost-of-living tracker that visualises how prices have changed across 193 countries over 1, 5, and 10-year windows. Choropleth world map, multi-category breakdowns (CPI, food, energy, education), country comparisons, and PPP-adjusted views — all powered by World Bank and FAOSTAT open data.

React TypeScript Leaflet Recharts World Bank API
View project
Status
Live
The Insight
Inflation is felt locally but driven globally. Most dashboards show one country at a time. This puts the whole world on one screen — compare your country against any other, across any category, in seconds.
What's Next
Personal basket weighting, threshold alerts, housing and healthcare categories, and historical event overlays on trend charts.
